ActiveClass JS

Usage :

Include the script : 
<script src="/path/to/activeclass-js.js"></script>
if your link have a parent element you can use attribute data-activeclassjs-parents="..." : 
<a href="..." data-activeclassjs-parents="group1"></a> (if you need you can use more then one parent. Use comma : group1,group2 ...)
and then add the attribute data-activeclassjs-ref="group1" on all parents: ` 
<div  data-activeclassjs-ref="group1">...</div>
